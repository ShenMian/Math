template<typename T, size_t N>
requires std::is_arithmetic_v<T>
class VectorT< T, N >

N 维向量.

模板参数
T数据类型.
N维数.

大小固定的向量, 可以用于表示位置/速度/方向等.

例子:

// 创建
Vector2i v(1, 2); // 创建一个 2 维向量, 类型为 int32_t, 初始值为 (1, 2)
// 操作
v.x = 3; // 设置 x 分量为 3
v[1] = 4; // 设置 y 分量为 4
std::cout << v.size() << std::endl; // 输出向量的长度, 结果为: 5
